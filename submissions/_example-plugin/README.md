# Example Plugin

> This is a reference plugin for contributors. Do NOT install it.

Demonstrates the standard submission format for Plugin Store community plugins.

## What it does

- Searches for tokens by name, symbol, or address
- Displays market prices for tokens

## Permissions

- ✅ Read balance
- ❌ Send transactions
- ❌ Sign messages
- ❌ Contract calls
